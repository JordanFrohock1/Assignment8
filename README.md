# FIZZ BUZZ ASSSIGNMENT 8

## What is this project?
#### This project is a common project for intertiwes for coding positions. Here is the breakdown of what the code is doing:

FizzBuzz is a beginner program used to practice loops and if/else statements.

The program counts through numbers, usually from 1 to 100.

For each number, it checks if the number is divisible by **3**, **5**, or both using the remainder (modulus) operator.

If the number is divisible by 3, it prints **Fizz** instead of the number.

If the number is divisible by 5, it prints **Buzz**, and if it is divisible by **both** 3 and 5, it prints **FizzBuzz**.

If none of the conditions are met, the program simply prints the original number.
